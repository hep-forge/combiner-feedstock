Feedstock for [combiner](https://gitlab.cern.ch/tdado/combiner) — part of [hep-forge](https://anaconda.org/hep-forge).
Builds linux-amd64 + linux-arm64 in one matrix workflow and uploads to the
[hep-forge](https://anaconda.org/hep-forge) Anaconda channel.

Combiner: statistical combination tool built on ROOT

## Architectures

State per architecture: see the platforms badge above or [anaconda.org/hep-forge/combiner](https://anaconda.org/hep-forge/combiner).


## Install

```bash
conda install -c hep-forge -c conda-forge combiner
```
